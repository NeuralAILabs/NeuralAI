# NeuralAI

NeuralAI is a Kathmandu-based company that combines AI with real teaching to make education more affordable and accessible. We build AI-powered software, run tech bootcamps and language coaching, and guide students through study abroad applications.

## What we do

- **IeltsBuddy** — AI-powered IELTS preparation platform for students targeting universities in the US, UK, Australia and Canada
- **Digital Sewa** — Nepal's local freelancing marketplace connecting clients with skilled service providers
- Tech certifications, IT bootcamps, and language coaching
- Study abroad guidance, scholarships, and visa support
